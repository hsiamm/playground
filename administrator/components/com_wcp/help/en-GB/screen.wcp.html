Working Copy

Description

This component will help you to create a working copy of your live site and track the made changes on it, so you can easily commit them to the live site.

Screenshot

Column Headers

  • #. An indexing number automatically assigned by Joomla! for ease of reference.

  • Checkbox. Check this box to select one or more items. To select all items, check the box in the column heading. After one or more boxes are checked, click a toolbar button to take an action on the selected item or items.

  • Name. Name of the child site.

  • Path. The relative path of the child site.

  • ID. This is a unique identification number for the child site. You can not change this number.

Toolbar

At the top right you will see the toolbar:

  • New. Opens a new child creation form.

  • Edit. Select one item and click on this button to open it in edit mode. You can also open an item for editing by clicking on its Name.

  • Delete. Select one item and click on this button to delete it. It will delete the child site tables and files.

  • Apply Patch. Opens a form for submitting a patch to the site.

  • Merge. Select 2 items and click on this button to merge the sites. The first item will be changed accordingly to include all the changes from the 2nd child site, while 2nd will stay intact.

  • Differences. This button will be available only on the child site. Click it and it will open the list of the made changes on the child.

  • Help. Opens this Help Screen.

Quick Tips

  • To start using this component begin with creating a new child.

  • To delete a child site use delete button, don't delete the files manually from the server, unless you know what are you doing.
